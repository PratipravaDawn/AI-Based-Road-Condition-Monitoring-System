# Road-Condition-Monitoring-System
An IoT-enabled embedded system that utilizes MPU6050 accelerometer data and machine learning algorithms to detect and classify road Good, Bad and Potholes, with GPS-based location tagging and wireless data transmission via ESP32.

## The system classifies road conditions into:
- Good Road 🟩
- Bad Road 🟥
- Pothole 🟣

## System Architecture:
MPU6050 Sensor → ESP32 → ML Processing → Bluetooth → Mobile App → Map Visualization

## Hardware Components
- ESP32 Microcontroller  
- MPU6050 Accelerometer & Gyroscope Sensor
- Power Supply (Battery + LM2596 + Switch)
