# AI-Based-Road-Condition-Monitoring-System
An IoT-enabled embedded system that utilizes MPU6050 accelerometer data and machine learning algorithms to detect and classify road anomalies such as potholes and speed breakers, with GPS-based location tagging and wireless data transmission via ESP32.
